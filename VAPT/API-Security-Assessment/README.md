# API Security Assessment - OWASP Juice Shop

## Overview

Security assessment of OWASP Juice Shop APIs using OWASP API Security Top 10 methodology.

## Tools Used
- Burp Suite Professional
- JWT.io
- Firefox Browser

## Methodology
1. Reconnaissance
2. API Discovery
3. Authentication Analysis
4. JWT Analysis
5. Authorization Testing
6. Reporting

## Vulnerabilities Identified

| Vulnerability | Severity |
|--------------|----------|
| Broken Authentication | High |
| Broken Object Level Authorization (BOLA) | High |
| Excessive Data Exposure | Medium |
| Sensitive Information Exposure | Medium |

- API Security Testing
- Burp Suite

- OWASP API Top 10
- Authorization Testing
