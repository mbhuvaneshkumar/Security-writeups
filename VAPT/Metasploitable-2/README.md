# Metasploitable 2 VAPT Assessment

## Overview
A full Vulnerability Assessment and Penetration Test conducted against the Metasploitable 2 Linux machine in an isolated VirtualBox lab environment.

## Tools Used
- Nmap
- Metasploit Framework
- VNC Viewer

## Methodology
1. Network Discovery
2. Service Enumeration
3. Vulnerability Analysis
4. Exploitation
5. Post Exploitation
6. Reporting

## Vulnerabilities Identified
| Vulnerability | Severity |
|--------------|----------|
| vsftpd Backdoor (CVE-2011-2523) | Critical |
| Samba Usermap Script RCE | Critical |
| Samba Symlink Traversal | Critical |
| VNC Weak Credentials | High |
| MySQL Weak Credentials | Critical |

- Vulnerability Assessment

- Exploitation
- Post Exploitation
