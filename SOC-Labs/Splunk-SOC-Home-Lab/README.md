# Splunk SOC Home Lab

This project demonstrates a SOC home lab built using Splunk Enterprise, Windows Security Logs, and Parrot OS.

## Scenarios Covered

- User Account Creation (4720)
- User Account Deletion (4726)
- Privilege Escalation (4732)
- Failed Login Detection (4625)
- Successful Login Detection (4624)
- RDP Monitoring
- PowerShell Monitoring
